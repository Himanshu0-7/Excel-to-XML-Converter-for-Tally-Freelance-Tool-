# Excel-to-XML-Converter-for-Tally-Freelance-Tool-
Excel to XML Converter for Tally – Freelance Project (2025) Tech: Python, Streamlit, Pandas  Built a tool to convert Excel bank statements into XML format for import into TallyPrime.  Developed a simple UI using Streamlit for file upload and automated XML generation.  Automated financial data transformation, reducing manual accounting work.
